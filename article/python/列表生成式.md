## 列表生成式

最常用的用法

```python
#初始列表
l = ['po8','pact','gai','jonyj','round2']

#先写一个函数
new_l = []
for name in l:
    if len(name) >= 4:
        new_l.append(name)
print(new_l)

#列表生成式就是上面代码的简洁写法
new_l2 = [name for name in l if len(name) >= 4]
print(new_l2)

输出:
['pact', 'jonyj', 'round2']
['pact', 'jonyj', 'round2']
```

## 字典生成式

```python
item = [('charles',666),('Mary',456),('Mike',651)]
dic = {k:v for k,v in item if v > 650}
```

## 生成器表达式

没有元组表达式 ，这个形式是生成器表达式

```python
g = (i for i in range(0,10,4) if True)
print(next(g))
print(next(g))
print(next(g))

输出:
0
4
8
```
