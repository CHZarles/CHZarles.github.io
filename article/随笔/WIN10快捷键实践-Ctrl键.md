## Ctrl + 字母

<!--more-->

Ctrl多和控制有关

1. Ctrl + O 打开文件管理器
2. Ctrl + N 打开此程序的新页面 (Maybe)
3. Ctrl + W 关闭当前程序当前页面(不管用就Alt+F4)
4. Ctrl + A  全选
5. Ctrl + C/X/V 复制/剪切/粘贴
6. Ctrl + Z 撤销操作 
7. Ctrl + D 删除



## Ctrl + Tap

当前程序内窗口之间切换



## Ctrl + Shift + N

在文件夹中新建文件夹

早浏览器中新建无痕浏览页面
