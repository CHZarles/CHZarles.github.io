## 异常处理

**一般程序出现异常会马上停止程序，进行异常处理可以捕捉异常同时避免程序因错误而中断**
**这里讨论的是如何捕捉那种“无法预知什么情况下会发生的错误”（也就是不能用if判断解决的)**
**程序越好 这种异常处理用得越少 换言之 这种语法用得越少越好**

```python
语法
try:
    子代码块  # 有可能会抛出异常的代码
except 异常类型 as e:
    pass
except 异常类型2 as e:
    pass
else:
    如果被检测的子代码块没有异常发生，则会执行else的子代码
finally:
    '''
        无论子代码块有无异常发生，都会执行finally的代码
    finally 可以和 try 单独使用
        try:
            ...
        finally:
            ...
    一般回收资源的操作都要放在finally
    
    '''

```

<!--more-->



### demo

```python
# demo
try:
    l = [1,2,3]
    print(l[3])
except (IndexError,NameError) as e:  #索引异常，名字异常
    print('异常信息:',e)
except KeyError as e:
    print('字典的key不存在:',e)
except  Exception as e: #万能异常
    print('所有异常都能匹配到')
else:
    print('没有异常')
```
