## 打印进度条

```python
# 打印进度条
# 先导知识
#左对齐%-s, 左对齐且固定宽度为%-50s，不够填空格
print('[%-50s]' %'>') # 输出[>
# 不换行且每次在行首打印
print('\r[%-50s]' %'>',end='')

import time

def proc_line(precent):
    if precent>1: precent = 1
    res = int(precent*50)*'#'
    print('\r[%-50s] %d%%' %(res,int(precent*100)),end='')


recv_size = 0
total_size = 23333

while recv_size < total_size:
    # 假设每次下载1024字节
    time.sleep(0.3)
    recv_size+=1024
    #打印进度条
    proc_line(recv_size/total_size)
```
