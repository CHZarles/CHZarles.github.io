**“生成器就是自定义的迭代器”**																																	

## 1.生成器

器即工具

所以故事要从函数说起

现在假设要做一个函数 可以返回多个返回值

```python
#定义如下函数 可以将yield看成return的代替品
def func():
    print("第一次")
    yield 1
    print("第二次")
    yield 2
    print("第三次")
    yield 3
    print("第四次")
    yield 4
    
#执行，返回一个生成器
res = func()
print(res)


输出:
<generator object func at 0x031AA450>

#generator即生成器
```

## 3.应用案例

**造一个range函数**

```python
def my_range(start, stop, step):
    while start + step < stop:
        start += step
        yield start


# 调用
for it in my_range(0, 10, 3):
    print(it)
 
输出:
3
6
9
```



## 4.Yield的高级用法

**yield的厉害之处在于可以将函数挂起**

yield的高级用法在于 ，可以给自己的生成器传参

```python
def eater():
    print('\nReady to eat')
    while True:
         food=yield
         print('get the food: %s, and start to eat' %food)


#调用函数构造出生成器
g = eater()

#生成器在开始时 不能send非none值
#相当于send(None)
next(g)

#send方法相当于在next方法基础上多了传值功能
#send在给yield赋值,yield再将值赋给food
g.send('包子') 
g.send('香蕉')

输出:

Ready to eat
get the food: 包子, and start to eat
get the food: 香蕉, and start to eat
```



考虑有返回值的情况

```python
def eater():
    last_food = None
    print('\nReady to eat')
    while True:
         if last_food ==None:
             print("这是第一次吃东西")
         else:
             print('上次吃了 %s' %last_food)
         
         food=yield last_food
         last_food = food
         print('get the food: %s, and start to eat' %food)



#调用函数构造出生成器
g = eater()
next(g)
a = g.send('包子')
b = g.send('香蕉')
print(a,b)

输出:
Ready to eat
这是第一次吃东西
get the food: 包子, and start to eat
上次吃了 包子
get the food: 香蕉, and start to eat
上次吃了 香蕉
包子 香蕉
```
