## 整数二分

### 二分查找的实质

二分查找的实质是把一个区间分成两部分，一部分满足性质，一部分不满足性质。（和单调性还是有区别的）

![image.png](https://gitee.com/chzarles/images/raw/master/imgs/006eb5E0gy1gb0gkdgwpmj30xq03ddfo.jpg)

有两种写法，一种写法是找不断压缩空间找 
<!--more-->



**找左边界**

**这个写法的要诀是，记住向左压缩，改r**

```c++
bool check(int x) {/* ... */} // 检查x是否满足某种性质

// 区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int bsearch_1(int l, int r)
{
    while (l < r)
    {
        int mid = l + r >> 1;
        if (check(mid)) r = mid;    // check()判断mid是否满足性质
        else l = mid + 1;
    }
    return l;
}
```



**找有边界**

**这个写法的要诀是，记住向右压缩，改l**

```c++
// 区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int bsearch_2(int l, int r)
{
    while (l < r)
    {
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```



## 实数二分

```c++
bool check(double x) {/* ... */} // 检查x是否满足某种性质

double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}
```
