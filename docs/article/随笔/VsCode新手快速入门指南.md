## 2.1.光标移动

```text
移动到行首 home
移动到行尾 End
移动到文档的开头和末尾  Ctrl+Home/End
回到光标上次的位置  Ctrl + u
```

## 2.2.删除、移动、注释

```text
复制光标行代码  Alt+shift+上下  
单行注释  Ctrl +/
块注释 Alt+Shift+A
删除代码   Ctrl+Shift+K
```

## 2.3.代码格式化

```text
 Alt+Shift+F
```
