### 快速排序

```c++
//模板  
//左闭右闭
//维护两个指针 i 指针左边全是小于等于mid, j 指针右边全是大于等于mid
void quick_sort(int q[],int l,int r)
{
    if(l>=r) return ;
    int i = l - 1 , j = r + 1 , mid = q[i+j>>1];
    while(i < j)
    {
        do i++; while(q[i]<mid);
        do j--; while(q[j]>mid);
        if(i<j) swap(q[i],q[j]);
    }
    //出来时,i>j,且 i  - j =  1
    //记住吧
    cout<<i<<" "<<j<<endl;
    quick_sort(q,l,j);
    quick_sort(q,j+1,r);
}

```

### 快排特点应用：[第k个数](https://www.acwing.com/activity/content/problem/content/820/1/) 

给定一个长度为n的整数数列，以及一个整数k，请用快速选择算法求出数列的第k小的数是多少。


#### 输入格式

第一行包含两个整数 n 和 k。

第二行包含 n 个整数（所有整数均在1~109109范围内），表示整数数列。

#### 输出格式

输出一个整数，表示数列的第k小数。

#### 数据范围

1≤n≤100000,
1≤k≤n

#### 输入样例：

```
5 3
2 4 1 5 3
```

#### 输出样例：

```
3
```


#### AC代码

因为每次排序都会把元素分成大于key和小于key两部分。所以只要筛选适当的部分进行排序即可。


```c++
#include<iostream>
using namespace std;
const int N = 1e5+5;
int a[N];
int q(int l,int r,int k)
{
    if(l>=r) return a[l];
    int  i = l-1,j=r+1,x=a[l];
    while(i<j)
    {
        do i++ ;while(a[i]<x);
        do j-- ;while(a[j]>x);
        if(i<j) swap(a[i],a[j]);
        else break;
    }
    
    int s1 = j-l+1;
    //在这里,分出来有效部分
    if(k<=s1) q(l,j,k);
    else q(j+1,r,k-s1);
}

int main()
{
    int n,k;
    cin>>n>>k;
    for(int i=1;i<=n;i++) cin>>a[i];
    printf("%d\n",q(1,n,k));
}
```




### 归并排序

```c++
//模板
//左闭右闭
void sort(int g[],int l,int r)
{
    if(l>=r) return ;
    int  mid = l+r>>1;
    sort(g,l,mid);
    sort(g,mid+1,r);
    
    //这里假设两边都是单调的
    int temp[N], k = 0 ,i = l, j = mid+1;
    while(i<=mid&&j<=r)
        if(g[i]<g[j])
            temp[k++] = g[i++];
        else
            temp[k++] = g[j++];
            
    while(i<=mid) temp[k++] = g[i++];
    while(j<=r)  temp[k++] = g[j++];
    
    
    for(int i = l , k = 0 ; i <= r; i ++ )
        g[i] = temp[k++];
        
}
```

### 归并排序应用: [逆序对的数量](https://www.acwing.com/activity/content/problem/content/822/1/)  

给定一个长度为n的整数数列，请你计算数列中的逆序对的数量。

逆序对的定义如下：对于数列的第 i 个和第 j 个元素，如果满足 i < j 且 a[i] > a[j]，则其为一个逆序对；否则不是。


#### 输入格式

第一行包含整数n，表示数列的长度。

第二行包含 n 个整数，表示整个数列。

#### 输出格式

输出一个整数，表示逆序对的个数。

#### 数据范围

1≤n≤100000

#### 输入样例：

```
6
2 3 4 5 6 1
```

#### 输出样例：

```
5
```


#### AC代码

归并排序的过程中，把对应数轴上的数分成两部分，左边的数和右边的数分别单调排序。总逆序对部分是 ：左边逆序对个数+右边逆序对个数+左右两边的数构成的逆序对。


```c++
#include<iostream>
using namespace std;

//要开long long
typedef long long ll;
const int N = 100010;
int a[N];
//左闭右闭
ll work(int g[],int l,int r)
{
    if(l>=r) return 0 ;
    int mid = l+r>>1;
    ll res = work(g,l,mid) + work(g,mid+1,r);
    int i = l , j = mid + 1 , k = 0 , temp[N];
    
    while(i<=mid&&j<=r)
        if(g[i]<=g[j])
            temp[k++] = g[i++];
        else
        {
            temp[k++] = g[j++];
            res += (mid-i+1);
        }
        
    while(i<=mid) temp[k++] = g[i++];
    while(j<=r) temp[k++] = g[j++];
    
    for(int i = l , k = 0 ; i <= r ; i++) g[i] = temp[k++];
    
    return res;
}

int main()
{
    int n;
    cin>>n;
    for(int i = 0 ; i < n ;i++) cin>>a[i];
    cout<<work(a,0,n-1);
}
```
