## 作用

快速地将有交集区间集合成一个大区间

![image.png](https://gitee.com/chzarles/images/raw/master/imgs/006eb5E0gy1ggxa2kwjz6j30n908k3zw.jpg)

<!--more-->



其实用模拟，贪心的思想就能想明白。

```c++
// 将所有存在交集的区间合并
void merge(vector<PII> &segs)
{
    vector<PII> res;

    //左端点从小到大排序，pair默认比较第一个数
    sort(segs.begin(), segs.end());
	//st记录当前维护的大区间的首段点，ed记录尾端点
    int st = -2e9, ed = -2e9;
    for (auto seg : segs)
        if (ed < seg.first)  //如果当前段的尾端点比下一段的头端点小，开始合并下一段
        {
            if (st != -2e9) res.push_back({st, ed});
            st = seg.first, ed = seg.second;
        }
        else ed = max(ed, seg.second); //否则，更新当前段的尾端点

    if (st != -2e9) res.push_back({st, ed});

    segs = res;
}
```
